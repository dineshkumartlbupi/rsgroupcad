# Geon Energy Clone Setup

The project has been set up with React and Tailwind CSS.
However, due to a temporary environment issue, dependencies may not have been installed automatically.

## Quick Start

1. Open your terminal in this directory:
   ```bash
   cd /Users/dineshkumar/Documents/geon-clone
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Project Structure
- `src/components/Navbar.jsx`: Navigation bar with sticky effect.
- `src/components/Hero.jsx`: Main hero section with background and animations.
- `src/components/Services.jsx`: Services/Solutions grid.
- `src/components/Footer.jsx`: Footer with links.
- `src/App.js`: Main layout assembling the components.
- `tailwind.config.js`: Tailwind configuration.
